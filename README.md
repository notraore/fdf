# FdF - Fil de Fer (Wireframe)

This is a school projet called Fdf

<h2>Introducing to:</h2>
<ul>
  <li>Pixel drawing</li>
  <li>RGB color</li>
  <li>Bresenham's line algorithm</li>
  <li>File Parcing</li>
  <li>Map creation</li>
  <li>MinilibX Framework (x11 school Framework)</li>
  </ul>

Usage : `make`<br />
`./fdf map/[yout map]`
